## Iteration 3: Polish and Test
To improve the design, I added spacing, softer colors, rounded boxes, and clear buttons so the tool would feel more polished. I also added a statistics section so the user can quickly see how many tasks are total, active and completed. 

I added reset functionality with a “Reset All” button and also a “Clear Completed” button so the user can start over without refreshing the page.

I tested edge cases like empty input, deleting tasks, clearing completed tasks, and resetting everything. I also checked what happens when there are no tasks, so I added an empty-state message.

If I got feedback from a classmate, I would say they were able to understand the tool, but clearer feedback buttons and the empty message made it easier to use. With more time, I would add due dates, drag-and-drop reordering, and maybe a dark mode.
