## Iteration 1: Structure and First Interaction 
For the HTML structure, I used a main container with sections for the title, task input, filter buttins, task list, and statistics. I wanted the layout to feel organized and easy to use, so I separated each part of the tool cleary.

My first piece of JavaScript was the code that adds a new task when the user clicks the "Add Task" button. I used getElementById() to select the input and button, then used addEventListener() to make the button respond to a click. 

If AI generated part of the code, I still had to understand how it connected to my HTML. I had to make sure the JavaScript was selecting the correct elements, reading the input value, and updating the page by adding a new list item.
