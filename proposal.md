# Mini Project #2 Proposal 

## What I'm building 
I am building an interactive daily to-do list web tool usinG JavaScript. 

## Who it's for/why 
This tool is for students or anyone who wants to stay organized during the day. It helps users keep track of tasks, mark them as completed, filter them, and reset the list without refreshing the page. 

## Core features 
- Add new tasks 
- Mark tasks as completed 
- Delete tasks 
- Filter tasks by all, active, or completed 
- Clear all tasks and reset the tool 
- Save tasks using localStorage so they stay after closing the tab
