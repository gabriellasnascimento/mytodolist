# My Daily To-Do List

## What my tool does 
This is an interactive web-based to-do list that helps users organize their daily tasks. Users can add tasks, mark them as completed, delete tasks, filter tasks by status, and reset the list without refreshing the page. The tool also uses JavaScript to update the page dynamically and localStorage to save tasks between sessions. 

## How to use it 
1. Type a task into the input box.
2. Click **Add Task** or press **Enter** to add it to the list.
3. Check the box next to a task to mark it as completed.
4. Click **Delete** to remove a task.
5. Use the **All**, **Active**, and **Completed** buttons to filter tasks.
6. Click **Clear Completed** to remove finished tasks.
7. Click **Reset All** to clear the entire list and start over.
